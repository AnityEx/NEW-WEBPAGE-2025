## Setup Instructions

1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the portfolio.
3. Customize the `style.css` file to change the appearance of the website.
4. Add your portfolio images to the `images` folder.
5. Modify the `script.js` file to add any desired interactivity.

## Usage

This portfolio website can be used to display your 3D projects and artworks. Feel free to modify the content and styles to fit your personal branding.
